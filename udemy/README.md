# 🧠 Quiz Game (Python OOP)

A simple command-line Quiz Game built using Python and Object-Oriented
Programming (OOP). The application presents True/False computer science
questions, accepts user input, checks answers, and displays the final
score.

## Features

-   Object-Oriented Programming (OOP)
-   True/False quiz
-   Score tracking
-   Final score summary
-   Modular structure

## Project Structure

